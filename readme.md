# Higher or Lower

Will the next card in the deck be higher or lower than the current card?  Note:  if the values are the same, it is a tie

#### Technology Used

- HTML
- CSS with grid containers
- Javascript
- Jquery
- API calls using AJAX
- Local Storage


#### Link to Project

   https://project1-virid-nine.vercel.app/
